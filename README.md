# 查分网站自动查询脚本

通过自动批量填充输入框自动查询，导出查询结果

## 安装

### 安装Python

Download Python[<img src="https://www.python.org/static/img/python-logo@2x.png" width="64" alt="Python">](https://www.python.org/downloads)

### 安装依赖库
```
pip install selenium
```

```
pip install pandas
```

```
pip install openpyxl
```

> [!tip]
> **镜像源**
```
https://pypi.tuna.tsinghua.edu.cn/simple
```

## 运行

运行Python文件`catchgui.py`

## 使用
- 输入查分网站网址
- 选择Excel文件
- 选择下载目录
- 输入输出文件名

> [!important]
> **适用于以下网站**
>
> 易查分：https://www.yichafen.com/
>
> 智慧查分：https://www.zhihuichafen.cn/ （未测试）
